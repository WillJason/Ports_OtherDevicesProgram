介绍
1）通过键盘输入密码，密码输入正确后播放音频（01-001.mp3）并开锁；成功触发后再次输入密码复位；
2）键盘上只有部分按键有效。按下有效按键时有蜂鸣声，按下其它按键无蜂鸣声；
3）密码支持1~60位，可以由除回格键、回车键以外的所有有效按键组成；
4）任意时刻，按下回格键或者删除键，清除所有输入状态；
5）连续10s无任何输入，清除之前所有输入状态；

复位方式
无
连线说明
1）控制器上的红黑线输出12V，直接接电磁锁（红线接电磁锁的红线、黑线接电磁锁的黑线）；
2）控制器上引出的PS母头接口用来接PS2键盘（已经做好接口，直接对插即可）；
3）控制器接12V电源；

参数配置（指难度、密码等）
1.控制器上电后30秒内输入配置密码：sdam20160116，进入密码配置模式，系统播放音频（系统进入密码配置模式，密码不得超过六十位，密码输入完毕后按下回车键确认，现在请输入）；
2.密码配置完毕后系统播放音频（密码配置完成，请测试）；
3.密码配置模式只有在控制器上电后30秒内才能进入，超过时间后输入配置密码无效；
其他事项
有效密码按键说明：
1.主键盘区：
1）数字0~9；
2）字母A~Z；
3）空格键；
2.小键盘区：
数字0~9；
加、减、乘、小数点；

关键词:STC12C5A60S2 PS_2 yx5200
